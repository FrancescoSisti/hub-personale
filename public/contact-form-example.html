Esempio Form di Contatto

Come utilizzare questa API

Questo è un esempio di integrazione dell'API per form di contatto. Lo script si collegherà automaticamente a qualsiasi form con la classe contact-form.

Endpoint API: /api/v1/contacts

Integrazione del codice

1. Include lo script

<script src="https://your-domain.com/contact-form-integration.js"></script>

2. Crea il form HTML

<form class="contact-form">
    <input type="text" name="name" placeholder="Nome" required>
    <input type="email" name="email" placeholder="Email" required>
    <textarea name="message" placeholder="Messaggio" required></textarea>
    <button type="submit">Invia</button>
    <div class="contact-message"></div>
</form>

3. Configurazione personalizzata (opzionale)

<script>
// Configurazione personalizzata
const customContactAPI = new ContactFormAPI({
    baseUrl: 'https://your-domain.com/api/v1',
    formSelector: '.my-contact-form',
    messageContainerSelector: '.my-message-container'
});
</script>

4. Uso programmatico

<script>
// Invio manuale
contactFormAPI.submit({
    name: 'Mario Rossi',
    email: 'mario@example.com',
    message: 'Ciao, sono interessato ai vostri servizi.'
}).then(result => {
    console.log('Risultato:', result);
});
</script>
